# Running the game

Linux/OSX/Windows

- [Install LÖVE](https://love2d.org/)
- [Download the game]()
- Double-click to run it

Windows

- Download:
    - [32 bit]()
    - [64 bit]()
- Extract files
- Run game.exe

Linux/OSX/Windows from source

- Download the source files
- Unzip, keeping the directory structure intact
- Run it with LÖVE. Example:
        $ love [folder name]
